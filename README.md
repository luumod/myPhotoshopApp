## 简易版图片加工软件

基于 opencv + Qt6.2.4 来完成的。

主要是为了巩固 Qt 与学习 OpenCV 来做的这个软件。

### 目前进度

* 图像基础操作
* 图像模糊操作
* 图像阈值化操作
* 图像形态学操作
* 图像连通块分析
* 图像轮廓检测
* 图像直方图操作

###  项目特色

* 功能齐全，基本涵盖了上述的所有操作
* 可视化操作，基于QSlider或者QComboBox，允许用户可以随时指定不同参数的不同数值，以便更好的观察某个操作的具体效果。
* 混合加工模式，允许对同一张图片执行可控制，可视化的混合操作（具体行为因人而异）。
